# LBPresenter_iOS

## Installation

You can add `LBPresenter` to an Xcode project by adding it as a package dependency.

1. From the **File** menu, select **Add Packages...**
2. Enter the URL of this repository: `https://github.com/LunabeeStudio/LBPresenter_iOS`
3. Choose the version or branch you want to install.
